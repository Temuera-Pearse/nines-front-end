# Horse Assets

Use shared animation frames plus per-horse skins.

Layout:

- `shared/idle`: lineup or resting frames.
- `shared/gallop`: primary running cycle.
- `shared/stumble`: reaction frames for stun or disruption.
- `shared/finish`: finish or runoff poses.
- `shared/shadow`: reusable ground shadow assets.
- `skins/horse-*`: horse-specific silks, accents, and optional overlays.

Alpha recommendation:

- Keep one shared animation rig.
- Differentiate horses mainly through skins and color identity.
